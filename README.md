# ShopStyle | Modern E-Commerce Frontend 🛍️

A sleek, responsive, and high-performance e-commerce storefront built with **React**, **Vite**, and **Tailwind CSS**. This project focuses on a premium user experience with a clean UI inspired by modern shopping platforms.

## ✨ Features

- **Product Catalog:** Visually appealing display of products with modern hover effects.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Intuitive Navigation:** Category-based browsing for Men, Women, Kids, and Beauty.
- **Shopping Cart & Wishlist:** UI components ready for state management and CRUD operations.
- **Fast Loading:** Built with Vite for near-instant hot module replacement (HMR).

## 🛠️ Tech Stack

- **Frontend Library:** React.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Icons:** Lucide React / Custom SVGs
